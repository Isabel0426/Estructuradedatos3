# Estructuradedatos3
¿Qué es un árbol binario? 
Una estrctura de datos en la que cada nodo tiene máximo 2 hijos, uno izquierdo y uno derecho, para que se agreguen números, verlos en orden y buscarlos fácilmente.

¿Cómo se implemento?
1 clase Nodo para representar cada elemento del árbol.
1 clase ArbolBinario para gestionar: agregar datos, recorrido y busqueda de valores

Funcionalidad: 
- Agregar datos de tipo número en el árbol
- mostrarlos inorden
- Buscar un número
- usa un menú interactivo en consola

Usar el programa:
Para compilar:javac ArbolBinario.java
Para ejecutar: java ArbolBinario
